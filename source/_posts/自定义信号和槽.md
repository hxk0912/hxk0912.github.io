## 例程

比如买家得到优惠券，卖家给商品打折这个过程。

*Buyer.h*
``` C++
signals:
    void reveice();
```
*Seller.h*
``` C++
public slots:
    void discount();
```
*MainWindow.h*
``` C++
public:
    Seller * sr;
    Buyer  * br;
```
*Seller.cpp*
``` C++
void Seller::discount()
{
    qDebug()<<"商品打折"<<endl;
}

```
*MainWindow.cpp*
``` C++
MainWindow::MainWindow(QWidget *parent)
    : QMainWindow(parent)
{
    sr=new Seller(this);
    br=new Buyer(this);

    connect(br,&Buyer::reveice,sr,&Seller::discount);

    emit br->reveice();

}
```
